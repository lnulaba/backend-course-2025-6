## Структура проекту

```

├── SearchForm.html     # Веб-форма для пошуку предметів

```

## Запуск проекту

1. Встановлення залежностей:
   ```bash
   npm install
   ```

2. Запуск у режимі розробки:
   ```bash
   npm run dev -- --host localhost --port 3000 --cache ./cache
   ```

3. Доступ до API:
   - Веб-форми: http://localhost:3000/RegisterForm.html
   - API документація: http://localhost:3000/docs
   - REST API: http://localhost:3000/inventory
